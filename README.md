# English Janala

English Janala is an interactive vocabulary learning web application where users can explore lessons, search words, view meanings, and listen to pronunciation. The app loads vocabulary dynamically using API and provides a clean learning interface.

## 🔗 Live Demo
https://farhansm01.github.io/English-Janala/

---

## 🚀 Features

- Lesson based vocabulary learning
- Dynamic API data loading
- Word pronunciation (speech synthesis)
- Search vocabulary
- Word details modal
- Synonyms display
- Loading spinner
- Responsive design

---

## 🛠️ Tech Stack

- HTML
- Tailwind CSS
- DaisyUI
- JavaScript
- REST API
- Speech Synthesis API

---

## 📦 Dependencies

- Tailwind CSS (CDN)
- DaisyUI
- Font Awesome
- Programming Hero API

## ⚙️ Run Locally

Clone the project

```bash
git clone https://github.com/farhansm01/English-Janala.git
```

Go to project directory

```bash
cd English-Janala
```

Open in browser

```bash
open index.html
```

OR just double click `index.html`
